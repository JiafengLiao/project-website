User manual

1. Registration(First time user)

First time user should choose the acount type and enter registration details.

2. Registration success

 

3. Login

Enter the User name adn password after register successfully.

 

4. The main menu

Pressing the gress button for enter the game session selection menu. The blue shopping cart button is for the shop where u can purchase ad set the cars.

 

5. Select session

Now you can select session you want to play. The session is able to choose only if the previous session gets at leat one star.

 

6. Answer the questions and pass session!!!

When the car meet obstacles, the question will pop up. Answer it in time(Around 5 seconds)!  

If you answer the question correctly in time, your car will pass the obstacles and speed up.

If not, the car will be trapped by the obstacle for a while and decrease the speed.

 

7. The report page

After the session, ou will receive a report page consist of your accuracy and all the questions you answered.

The questions with green font colour indicates the questionsi answered correctly and red is not.

 

8. The shop(garage)

You can buy a car in the shop and set it to your default after purchsing.
